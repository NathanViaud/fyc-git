title: "Exemple 1 : Initialisation et premier commit"
---
import { Steps } from '@astrojs/starlight/components';

<Steps>
1. Initialiser un dépôt Git
    ```shell
    git init
    ```
    Cette commande initialise un nouveau dépôt Git dans le répertoire courant.

2. Ajouter des fichiers
    ```shell
    git add index.html
    git add style.css
    ```
    Ces commandes ajoutent `index.html` et `style.css` à l'index (staging area).

3. Faire un commit
    ```shell
    git commit -m "Ajout des fichiers initiaux du site web"
    ```
    Cette commande enregistre les modifications dans le dépôt avec un message descriptif.
</Steps>
